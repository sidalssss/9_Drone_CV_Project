# Drone Computer Vision Project

Bu proje, İnsansız Hava Araçları (İHA) için özelleştirilmiş nesne algılama ve takip algoritmalarını içermektedir.

## Özellikler
- **Nesne Takibi:** CSRT ve KCF algoritmaları ile hareketli nesnelerin sürekli izlenmesi.
- **Engel Algılama:** Görüntü işleme teknikleriyle drone rotası üzerindeki engellerin tespiti.
- **Navigasyon Desteği:** Görsel verilerle otonom uçuş güvenliğinin artırılması.

## Teknolojiler
- Python
- OpenCV (Tracking API)
- Real-time Image Processing
